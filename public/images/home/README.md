# Images nécessaires pour la page d'accueil

Placez toutes les images suivantes dans ce dossier (`/public/images/home/`).

## Images requises

### 1. Hero Section
- **hero-martinique.jpg** - Paysage immersif mer + relief (format large, recommandé 1920x1080px minimum)

### 2. Immersion Gallery
- **immersion-costumes-traditionnels.jpg** - Costumes traditionnels / danse
- **immersion-marche-local.jpg** - Marché local / vie quotidienne
- **immersion-architecture-creole.jpg** - Maisons créoles / patrimoine architectural
- **immersion-musique-tambours.jpg** - Musique / tambours traditionnels

### 3. Séjours Immersifs
- **sejour-immersif-village.jpg** - Vie locale / village / ruelles (format large, recommandé 1200x800px)

### 4. Piliers de l'immersion
- **pilier-culture.jpg** - Représentation de la culture martiniquaise
- **pilier-histoire.jpg** - Représentation de l'histoire
- **pilier-patrimoine.jpg** - Représentation du patrimoine
- **pilier-langue.jpg** - Représentation de la langue créole

### 5. Romans
- **romans-lecture-voyage.jpg** - Livre + paysage / lecture (format large, recommandé 1200x800px)

### 6. Témoignages
- **temoignage-voyageur-1.jpg** - Photo/avatar du premier témoin
- **temoignage-voyageur-2.jpg** - Photo/avatar du deuxième témoin
- **temoignage-voyageur-3.jpg** - Photo/avatar du troisième témoin

### 7. CTA Final
- **contact-voyageurs-plage.jpg** - Image discrète en fond (optionnel, format large)

## Format recommandé
- **Format** : JPG ou WebP
- **Taille** : Optimisées pour le web (max 500KB par image)
- **Dimensions** :
  - Images hero : 1920x1080px minimum
  - Images de section : 1200x800px
  - Images de galerie : 800x600px
  - Avatars témoignages : 200x200px (carré)

## Note
Les images sont référencées dans le code mais peuvent être remplacées facilement en modifiant les chemins dans `app/[locale]/page.tsx`.
